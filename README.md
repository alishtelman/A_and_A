# Свадебный лендинг (Алишер & Аяулым)

## Что уже готово
- Двухъязычный сайт: RU/KZ переключается кнопкой вверху.
- Анимации: плавное появление блоков, декоративные фоновые элементы.
- Блоки: приглашение от родителей, место/время, таймер, программа, галерея, RSVP.

## Что заменить перед отправкой
1. Дата тоя:
- Файл: `script.js`
- Поле: `config.eventDateISO`
- Формат: `2026-07-18T16:00:00+05:00`

2. Место и время:
- Файл: `script.js`
- Блоки в `texts.ru` и `texts.kz`: `whereValue`, `timeValue`

3. Ссылка на карту:
- Файл: `index.html`
- Строка с `data-i18n="mapLink"` (при необходимости заменить ссылку 2GIS)

4. WhatsApp для подтверждения:
- Файл: `script.js`
- Поле: `config.whatsappNumber` (введите номер в международном формате без `+`)

5. Фоновая музыка:
- Файл: `assets/toi-shashu.mp3`
- В шаблоне уже подключен плеер в шапке. Просто положите трек с таким именем в папку `assets`.

5. Фото:
- Замените файлы:
  - `assets/photo-1.svg`
  - `assets/photo-2.svg`
  - `assets/photo-3.svg`
- Можно заменить на `.jpg/.png`, тогда поправьте расширения в `index.html`.

## Быстрый запуск
Откройте `index.html` в браузере.
